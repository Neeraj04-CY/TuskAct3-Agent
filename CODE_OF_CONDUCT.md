# Code of Conduct

We are committed to a professional, inclusive community. By participating in this project you agree to:

1. Be respectful and constructive in all discussions and reviews.
2. Assume good intent, ask clarifying questions, and avoid personal attacks.
3. Keep private information (API keys, customer data, credentials) out of the repository and public channels.
4. Report unacceptable behavior to the maintainers at security@example.com so we can respond quickly.

Maintainers may remove content or restrict access to preserve a safe environment. Thank you for helping make EIKON Engine a welcoming project.
